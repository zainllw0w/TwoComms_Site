# 🤖 Как работает CRON - Простое объяснение

## ❓ Вопрос: Нужно ли запускать CRON вручную?

### ✅ ОТВЕТ: НЕТ! CRON работает автоматически

**CRON** - это система на сервере, которая **сама запускает команды по расписанию**.

---

## 🔄 Как это работает:

### 1. **Вы настраиваете ОДИН раз** (через cPanel или SSH)

### 2. **Сервер САМ выполняет команду** по расписанию:
   - Каждые 5 минут ✅
   - Каждые 15 минут ✅
   - Каждый час ✅
   - И т.д.

### 3. **Вы ничего не делаете** - всё работает автоматически!

---

## 📊 ТЕКУЩЕЕ СОСТОЯНИЕ ВАШЕГО СЕРВЕРА

У вас **УЖЕ настроен** cron для Nova Poshta:

```
*/15 * * * * ... update_tracking_statuses ...
```

Это означает: **работает каждые 15 минут автоматически**

---

## ⚠️ ЧТО НУЖНО ИЗМЕНИТЬ

Измените с **15 минут** на **5 минут**:

### Через cPanel:

1. Зайдите в **cPanel → Cron Jobs** (или "Запланированные задачи")
2. Найдите задачу с `update_tracking_statuses`
3. Измените **Минута:** с `*/15` на `*/5`
4. Сохраните

### Или через SSH (если нужно):

```bash
sshpass -p '[REDACTED_SSH_PASSWORD]' ssh qlknpodo@195.191.24.169 "bash -lc 'crontab -l | sed \"s|*/15|*/5|g\" | crontab -'"
```

Но **проще через cPanel** - там видно всё в интерфейсе!

---

## 📝 ИНСТРУКЦИЯ ДЛЯ CPANEL

### Шаг 1: Найти Cron Jobs

1. Войдите в **cPanel**
2. Найдите **"Cron Jobs"** или **"Запланированные задачи"**
   - Обычно в разделе "Дополнительно" или "Advanced"

### Шаг 2: Изменить существующую задачу

1. Найдите в списке задачу с текстом: `update_tracking_statuses`
2. Нажмите **"Изменить"** или **"Edit"**
3. В поле **"Минута"** замените `*/15` на `*/5`
4. Нажмите **"Сохранить"**

### Или добавить новую (если старой нет):

**Поля:**
```
Минута:  */5
Час:     *
День:    *
Месяц:   *
Неделя:  *
```

**Команда:**
```bash
cd /home/qlknpodo/TWC/TwoComms_Site/twocomms && /home/qlknpodo/virtualenv/TWC/TwoComms_Site/twocomms/3.13/bin/python manage.py update_tracking_statuses >> /tmp/nova_poshta_update.log 2>&1
```

## ✅ ПОСЛЕ НАСТРОЙКИ

### Всё работает автоматически:

- ✅ Сервер сам проверяет статусы каждые 5 минут
- ✅ Не нужно ничего запускать вручную
- ✅ Не нужно подключаться по SSH
- ✅ Работает 24/7

### Что происходит каждые 5 минут:

1. Система проверяет все заказы с ТТН
2. Получает статусы через API Новой Почты
3. Когда посылка получена:
   - Меняет статус на "Отримано"
   - Меняет оплату на "Оплачено"
   - Отправляет в Meta Pixel
   - Отправляет уведомления в Telegram
