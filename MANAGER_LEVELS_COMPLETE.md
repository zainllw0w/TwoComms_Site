### 2. Запустити тестовий скрипт
```bash
cd ..
python test_manager_levels.py
```

### 3. Призначити рівень менеджеру
```python
python manage.py shell

from django.contrib.auth import get_user_model
from management.services.manager_levels import promote_manager
from decimal import Decimal
from datetime import date

User = get_user_model()
user = User.objects.get(username='manager')
admin = User.objects.filter(is_staff=True).first()

promote_manager(
    user=user,
    new_level='level_1',
    promoted_by=admin,
    weekly_salary=5000,
    commission_percent=Decimal('5.5'),
    start_date=date.today(),
    comment='Призначення'
)
```

### 4. Перевірити профіль
Відкрити: `http://management.twocomms.shop/profile/`
