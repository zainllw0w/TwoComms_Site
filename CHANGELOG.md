# CHANGELOG

## 1) Что было сделано (по блокам)

### Хедер / навигация
- Хедер стабилизирован в одну линию, контакты через иконки с hover/tap, профиль через аватар + мини-панель.
- Статусная длинная строка вынесена в top-bar.
- Добавлены поясняющие title для маршрутов «Конструктор» и «Вироби» как «под ключ».

### Главная / hero
- Удержана иерархия CTA: 1 primary в зоне hero, вторичные/ссылочные CTA рядом.
- Добавлен micro-offer «Потрібно під ключ?» без смещения фокуса с DTF.
- Copy по sample-предложению унифицирован в «тест‑пак» вместо акцента на A4.

### CTA‑система
- Зафиксирована CTA-иерархия в стилях: primary / secondary / tertiary.
- Повторяющиеся CTA получили менее агрессивный визуальный вес и контекстный copy.

### Локализация
- Убраны остатки debug/служебного UI в целевых страницах O3.
- Исправлены остатки EN/неоднородного copy в P0/P1 критичных блоках (price/sample/faq/gallery и т.д.).
- Исправлены ошибочные `elsif` в `gallery.html`.

### Preflight / QC
- Preflight-статусы и risk-ack унифицированы (OK/INFO/WARN/FAIL).
- На QC-странице добавлен before/after compare-слайдер и усилен процессный copy доверия.

### Зразок (Tech‑Pack)
- Страница перепозиционирована в «тест‑пак / Tech‑Pack».
- Форма переведена в progressive disclosure: лёгкий первый шаг + раскрытие дополнительных полей.
- Доставка и формат раскрываются прозрачно в деталях.

### Вимоги / Шаблони
- Добавлен быстрый чек‑лист готовности макета с tooltip-пояснениями.
- Добавлены блоки «OK vector / risk raster», Safe area-схема, «типовые ошибки».
- Блок шаблонов 60 см перестроен: 1 primary CTA + форматы + короткая инструкция в 3 шага.

### Цены / калькулятор / метраж
- Явно объяснён метраж как длина файла в метрах при ширине 60 см.
- Добавлен мини-калькулятор-ориентир (длина/копии -> метраж/ориентировочная цена).
- Добавлен B2B микро-CTA для цехов/опта (условия/SLA).

### Mobile UX
- Добавлен sticky mobile bottom bar под 4 ключевых действия.
- Добавлен mobile contact sheet (Telegram / телефон / помощь с макетом).
- Сохранён учёт `prefers-reduced-motion` и адаптивная деградация эффектов.

### SEO / Performance / A11y
- Добавлен schema `Organization` в базовый шаблон.
- На ключевых внутренних страницах сохранены/добавлены `BreadcrumbList` и FAQ schema.
- Tooltip и интерактивы приведены к keyboard/focus-friendly паттерну в рамках текущего UI.
