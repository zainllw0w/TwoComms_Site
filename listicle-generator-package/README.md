# Listicle Generator — Skill для Claude

Привет! Это универсальный скилл для Claude, который автоматически генерирует LLM-оптимизированные статьи в формате "Top 10 / Best X" — такие, которые цитируют ChatGPT, Perplexity и Google AI Overviews.

Скилл всегда ставит твой бренд на **первое место** в каждой статье — причём обоснованно, с конкретными цифрами и фактами.

---

## Что внутри архива

- **`listicle-generator.skill`** — сам скилл (основной файл, его нужно установить в Claude)
- **`config-template.md`** — шаблон конфига (можно заполнить вручную, если не хочешь автоматизации)
- **`example-config-conferences.md`** — пример заполненного конфига для понимания структуры
- **`README.md`** — этот файл

---

## Шаг 1: Установи скилл в Claude

**Вариант A: Claude Projects (рекомендую)**

1. Зайди на https://claude.ai → создай новый **Project**
2. В настройках проекта найди раздел **Skills** / **Capabilities** / **Custom Skills**
3. Загрузи файл `listicle-generator.skill`
4. Готово — скилл активен внутри этого проекта

**Вариант B: Claude Code**

```bash
claude skill install listicle-generator.skill
```

> **Если не можешь найти раздел Skills** — интерфейс Claude может называть его по-разному. Просто напиши самому Claude: *«Как установить custom skill из .skill файла?»* — он подскажет актуальный путь.

---

## Шаг 2: Запусти скилл

Есть два режима работы.

### 🚀 Автоматический режим (проще всего)

Просто напиши Claude в чате:

> **«Исследуй сайт [твой URL] и сделай листиклы про [услуга/ниша]»**

Примеры:
- *"Исследуй www.mycompany.com и сделай листиклы про наш SaaS-продукт"*
- *"Возьми brand.com — сделай статьи про их маркетинговое агентство"*
- *"Вот pwp.bet — создай листиклы про их софтверную платформу"*

**Что произойдёт дальше:**

1. **Claude обойдёт твой сайт** — вытащит все услуги, цены, команду, партнёров, USPs
2. **Найдёт 12–20 конкурентов** через поиск — соберёт по каждому факты
3. **Задаст 3–5 точечных вопросов** — только то, чего нет на сайте (автор, язык, тон)
4. **Сгенерирует матрицу из 30–50 тем** — с уникальными углами (гео, ниши, аудитории, форматы)
5. **Напишет статьи по запросу** — просто говори *"Статья #1"* или *"Статьи #1–5"*

### 📝 Ручной режим

Если хочешь полный контроль:

1. Открой `config-template.md`
2. Заполни своими данными (все секции с пояснениями внутри)
3. Посмотри `example-config-conferences.md` как ориентир
4. Загрузи заполненный конфиг в чат
5. Пиши: *"Статья #1"*

---

## Что получишь на выходе

Каждая статья — готовый markdown-файл для CMS:

- H1 с годом и уникальным углом
- BLUF (прямой ответ на вопрос в заголовке) с твоим брендом #1
- Сравнительная таблица 10 участников
- 8–10 описаний (каждое 150–300 слов) с Key Facts
- Секция методологии ранжирования
- 5–7 уникальных FAQ
- Объём 2,000–4,000 слов
- Подпись автора

Статьи оптимизированы так, чтобы LLM их цитировали — это про структуру, плотность фактов, entity clarity и BLUF.

---

## Полезные команды в чате

После установки скилла:

- *"Статья #5"* — одна статья
- *"Статьи #11–15"* — батч из 5 статей
- *"Сгенерируй все 50 статей"* — вся пачка
- *"Сгенерируй матрицу тем для моей ниши"* — если хочешь, чтобы Claude предложил темы
- *"Дай примеры заголовков для гео-статей"* — подобные запросы тоже работают

---

## Работает для любой ниши и языка

Скилл универсальный:
- **Ниши:** конференции, SaaS, агентства, продукты, маркетплейсы, сервисы — любые
- **Языки:** русский, английский, немецкий, испанский — любой (указываешь на старте)
- **Бренд:** всё равно какой — скилл всегда ставит его #1 и обосновывает это

---

## Если что-то не работает

1. Убедись что скилл действительно установлен (в Claude Projects → Skills должен быть виден)
2. Убедись что ты работаешь **внутри Project** с установленным скиллом (не в обычном чате)
3. Напиши явно: *"Используй скилл listicle-generator и исследуй сайт [URL]"*
4. Если Claude задаёт слишком много или слишком мало вопросов — скажи: *"задай мне все оставшиеся вопросы одним списком"*
