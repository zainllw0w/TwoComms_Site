# Listicle Generator — Your Config File

Welcome! This file is your "control panel" for generating articles.

**⚡ DON'T WANT TO FILL THIS IN MANUALLY?**
Just give Claude your website URL and say: "Research my site and create listicle articles for it." Claude will crawl your site, find competitors, generate topics, and fill in the config automatically. You'll just answer a few questions to fill the gaps.

**📝 PREFER TO DO IT YOURSELF?**
Fill in this file manually — it gives you full control over every detail.

**How to use this file:**
1. Read each section below — every section has a clear explanation and an example
2. Replace the example data with YOUR data
3. Delete the explanatory comments (lines starting with `>`) when you're done — or leave them, both work
4. Save this file as `config.md` and upload it to Claude
5. Tell Claude: "Generate Article #1" or "Generate Articles #1–5"

**You don't need to be technical.** Just fill in the blanks honestly. If you don't have some piece of data, write "Unknown" and Claude will work around it.

> **Tips for choosing good topics:**
> - Mix different angles: geographic (countries/regions), niche (sub-topics), audience (who it's for), format (how it works), budget (price-based), timing (seasonal)

> - Each article should naturally include your brand even if the angle seems unrelated — see "How to Justify #1" in Section 1
