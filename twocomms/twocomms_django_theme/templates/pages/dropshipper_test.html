Инструкции по использованию

  1. Добавьте дропшип цены в товары: В админке Django добавьте поля drop_price и recommended_price в модель Product
  2. Создайте пользователя-дропшипера: Зарегистрируйте пользователя и назначьте ему права доступа
  3. Примените миграции: Выполните python manage.py migrate для создания таблиц в базе данных
  4. Настройте URL маршруты: Убедитесь, что маршруты orders/ включены в основной urls.py
  5. Протестируйте функциональность: Перейдите по ссылкам выше для тестирования каждой части системы
