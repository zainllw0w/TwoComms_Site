## Как использовать

### Agent3 (Gemini 3.1) — Исполнитель-интегратор
1. **Первым делом** читает `ITER3_GLOSSARY.md`, `ITER3_FILE_MAP.md` и **`ITER3_AGENT4_HANDOVER.md`**.
2. Внедряет тексты, HTML/JS логику и подключает готовые SVG.
3. Правит dot bg + UI, опираясь на готовый CSS от Agent4.
4. В конце: `git push` + `collectstatic` + деплой.

### Agent4 (Gemini 3.1) — Дизайнер-создатель
1. Читает файл мапы проекта.
2. Генерирует 15 SVG-иконок и CSS анимации. Не трогает backend, logic, HTML файлы.
3. По завершению **создаёт `ITER3_AGENT4_HANDOVER.md`** — отчёт какие CSS-классы и файлы он подготовил. (Это служит флагом "можно работать" для Agent 3).

### Порядок запуска
```
Agent4 (SVG + CSS) ──(Writes HANDOVER)──→ Agent3 (HTML + тексты + dot bg) ──→ Deploy
```
