# ITER2_05 — UI Micro‑Pack (мікро‑графіка + мікро‑анімації)

Ціль: **додати “вау” і довіру**, але не вбити швидкість. Усе — максимально CSS‑only або мінімальний vanilla JS.

## 0) Загальні правила (обовʼязково)
1) Усі анімації мають поважати `prefers-reduced-motion: reduce` (вимикаємо/спрощуємо).  
2) Ніяких важких canvas/WebGL на мобілках у P0.  
3) Жодних анімацій, які конкурують з головною CTA‑кнопкою (замість цього — підсвітка “підказує куди тиснути”).  
4) Один “wow‑ефект” на екран максимум.

---

## P0 (впровадити разом з текстами)

### 1) “Лампочка” для меседжу про світло/генератор
**Де:** у блоці переваг / довіри (не в hero).  
**Сенс:** візуально підкреслити “працюємо без пауз” без слів про “проблеми з підрядниками”.

**Мікро‑графіка:** маленька line‑іконка лампочки (SVG).  
**Анімація (CSS):**
- раз на 6–8 сек короткий “blink” (200–300ms) + легкий glow,
- при hover/focus (desktop) — “on” (плавне світіння),
- на мобілці — вмикаємо glow, коли карточка потрапляє в viewport (IntersectionObserver), 1 раз.

**Текст поруч (UA):** `Є генератор — працюємо без пауз`  
*(або мʼякше: `Працюємо стабільно (є генератор)`)*.

---

### 2) “Сканування файлу” для блоку «Перевірка файлу»
**Де:** /order/ + /constructor/app/ (де є перевірка).  
**Мікро‑графіка:** іконка файлу (SVG) + тонка лінія‑сканер.

**Анімація:**
- Поки статус `Перевіряємо файл…` — лінія рухається зверху вниз 1.2–1.6s, easing `linear`.
- Коли OK — короткий “tick” (checkmark draw) 250ms.
- Коли Warn — “pulse” жовтого контуру 2 рази (не червоний).

**Важливо:** не робити “страшних” червоних мигалок.

---

### 3) Secondary CTA “Безкоштовний тест” — спокійний “breathing” pulse
**Де:** hero і внизу /quality/.  
**Анімація:** раз на 10–12 сек легке збільшення 1.00 → 1.02 → 1.00 (600ms), opacity border 0.8→1.0.  
**Мета:** показати опцію безкоштовного тесту, але не змагатися з primary.

---

### 4) Мікро‑підсвітка “Telegram” у mobile dock
**Де:** мобільний dock.  
**Анімація:** при першій появі dock — 1 раз “glow” іконки Telegram 400ms.  
**Мета:** підсвідомо показати “ось де швидко запитати”.

---

## P1 (коли дійдете до калькулятора та мобільного sticky)

### 5) Sticky CTA Bar (контекстна)
**Де:** мобільні сторінки після скролу ~35% (home/price/requirements/quality).  
**Вигляд:** тонка панель над dock: короткий текст + 1 кнопка.

**Приклади текстів:**
- Home: `Готові? Замовляйте друк →`
- Price: `Надіслати розміри — порахуємо →`
- Requirements: `Сумніваєтесь у файлі? Перевіримо →`

**Анімація:** slide‑up 200ms, без bounce.

---

### 6) “Value framing” у калькуляторі (ціна за 1 принт)
**Де:** /price/ таб “За розмірами”.  
**Мікро‑анімація:** коли змінюються значення — цифри “flip” без 3D (просто fade‑out/in 120ms) або count‑up 250ms.

---

## P2 (тільки якщо потрібен додатковий вау)

### 7) “Peel & reveal” (скролл‑анімація плівки)
**Ризик:** дорого по часу, легко вбити FPS на Android.  
**Як робити безпечно:** тільки на desktop, тільки 1 секція, з fallback static на мобілці.

---

## Додатково: міні‑іконки (без переробки дизайну)
Набір з 6–8 SVG іконок у одному стилі (тонкі лінії, rounded caps):
- файл / перевірка
- плівка / лист 60 см
- чистий край
- колір (палітра/градієнт)
- доставка (НП/посилка)
- лампочка/генератор
- Telegram

**Правило:** одна товщина лінії + одна система розмірів (16/20/24px).
