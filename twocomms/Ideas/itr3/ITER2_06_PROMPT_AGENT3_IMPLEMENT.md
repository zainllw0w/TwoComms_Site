# ITER2_06 — PROMPT для 3‑го агента (исполнитель правок сайта)

Ты — AI‑агент‑исполнитель. У тебя есть полный доступ к репозиторию/серверу dtf.twocomms.shop (Django + vanilla JS/CSS), ты видишь структуру шаблонов, локализации, анимации и можешь править код.  
Твоя задача — **аккуратно и быстро** внедрить Iter2: заменить тексты, убрать англицизмы/жаргон, слегка улучшить UI в рамках микропака, не ломая текущий дизайн и не ухудшая скорость.

## 0) Жёсткие ограничения (обязательно)
1) **Не меняй архитектуру страниц** и не делай глобальный редизайн. Только тексты + небольшие безопасные UI‑правки.
2) **Не добавляй тяжёлые библиотеки** и зависимости.
3) Все новые анимации — только CSS/минимальный JS и обязательно `prefers-reduced-motion`.
4) **Не добавляй фейковые метрики/таймеры/дефицит.** Если показываем цифру — она должна быть правдой.
5) Никаких “красных” тревожных предупреждений на hero.
