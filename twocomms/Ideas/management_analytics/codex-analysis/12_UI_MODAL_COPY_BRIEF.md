# UI and Modal Copy Brief

## 1. Цель
Дать короткий пакет текста для предупреждений и подсказок в ключевых точках workflow.

## 2. Warning copy snippets

### 2.1 Not interested without reason
"Ви вибрали 'Не цікаво' без причини. Без причини обробка майже не зараховується в ефективність."

### 2.2 Other reason without details
"Для причини 'Інше' додайте коротке пояснення, інакше запис буде неякісним для статистики."

### 2.3 Sent email without CP link
"Оберіть конкретне КП зі списку відправлених листів. Без цього статус не буде верифіковано."

### 2.4 Callback missing date
"Вкажіть дату та час передзвону, щоб клієнт не загубився в обробці."

### 2.5 Duplicate detected
"Цей контакт вже є в базі. Перевірте останню обробку перед створенням нового запису."

## 3. Manager digest tone
- Формат: коротко, фактично, з однією головною дією на день.
- Заборонено: розмиті фрази без evidence.

## 4. Admin digest tone
- Формат: ризики, KPI-відхилення, no-report флаги, пріоритетні менеджери для перевірки.

## 5. Color semantics
- Green: stable/good.
- Yellow: caution.
- Orange: risk.
- Red: critical.
