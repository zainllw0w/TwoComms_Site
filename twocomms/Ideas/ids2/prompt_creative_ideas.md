## Контекст проекта

### Бизнес-модель
**dtf.twocomms.shop** — это суб-домен компании TwoComms, специализирующийся на DTF-печати (Direct-to-Film). Это технология переноса изображений на текстильную продукцию через специальную плёнку.

**Основные продукты**:
1. **DTF-плёнка** (основной) — печать на рулоне шириной 60 см. Клиент присылает файл (макет), мы печатаем плёнку, которую клиент переносит на свою продукцию прессом. Ценообразование: по метрам (350–280 грн/м, скидки от объёма).
2. **Готовый мерч** (дополнительный) — печать + перенос на готовые бланки (футболки, худи). Клиенту отправляется готовая продукция.

**Стадия**: Pre-launch. 0 продаж, 0 отзывов. Сайт полностью готов технически, но не запускался коммерчески.

**Целевые сегменты**:
- **B2B цеха** — DTF-производства, которые заказывают плёнку оптом (50–100+ м/мес)
- **B2B бренды** — бренды одежды, streetwear, которые заказывают принты для своей продукции
- **B2B корпоративный мерч** — компании для event/team merch
- **B2C** — физические лица, которые хотят готовую продукцию (подарок на ДР, индивидуальный дизайн)

### Каналы трафика (планируемые)
1. **Instagram Ads** — холодный трафик, ~85% мобильный
2. **Google Ads** — тёплый/горячий трафик, поисковые запросы
3. **Органика** — SEO (Knowledge Base / Blog)
4. **Telegram** — основной канал коммуникации с клиентами. Большинство заказов предполагается через Telegram.

### Технический стек
- Django (Python) — бэкенд
- Jinja2/Django Templates — фронтенд
- HTMX — dynamic interactions
- Vanilla CSS (custom design tokens)
- Self-hosted fonts
- Три языка: UA (основной), RU, EN
- Хостинг: shared hosting (без Redis/Celery)

## Твоя задача

Сгенерируй **максимально полезные, креативные и реалистичные** идеи для улучшения сайта. Каждая идея должна:

1. **Давать реальный буст** — конверсии, доверию, UX, SEO или визуальному впечатлению
2. **Быть реализуемой** на текущем стеке (Django + vanilla CSS + JS)
3. **Подходить для pre-launch стадии** (0 продаж, 0 отзывов)

#### 1. Текст и именование
- Какие термины/фразы переименовать? (помимо уже решённых «preflight», «ганг-ліст», «тест-пак»)
- Какие заголовки/CTA переформулировать?
- Тон голоса: какой лучше для холодного Instagram-трафика?

#### 2. CTA-кнопки и формы
- Какие кнопки добавить/убрать/изменить?
- Оптимальные формулировки CTA?
- Как уменьшить friction в формах?

#### 3. Цвет и визуальные элементы
- Текущая палитра: тёмный фон (#050505), оранжевые акценты, amber-тона
- Какие цветовые решения улучшат конверсию?
- Где добавить визуальные акценты?

#### 4. Анимации и WOW-эффекты
- Какие новые анимации добавить? (помимо текущих 10 эффектов)
- Где микроанимации увеличат engagement?
- Какие «вау»-моменты создать для Instagram-аудитории?

#### 5. Элементы конверсии

### Ограничения

- **Не предлагай фейковые отзывы/числа** — компания pre-launch
- **Не предлагай смену стека** — Django + vanilla CSS + JS зафиксированы
- **Не предлагай A/B тесты** — пока не готовы
- **Не предлагай то, что уже решено** (список выше)
- **Акцент на Instagram-first** — мобильный трафик, быстрые решения, визуальная привлекательность
- **Telegram — основной канал конверсии** — каждая идея должна учитывать, что основная цель — привести человека в Telegram-чат

### Критерий качества

Идеи должны быть такими, чтобы мы могли передать каждую напрямую разработчику и он знал **что именно, где именно и зачем** нужно сделать. Абстрактные рекомендации типа «Улучшить UX» — не принимаются.
