# Редизайн `custom-print` в лид-форму

## Цель
- Убрать перегруженный лендинг и превратить страницу в понятную вертикальную форму-заявку.
- Сделать основной сценарий простым: пользователь быстро выбирает услугу, изделие, зоны принта и канал связи, после чего заявка уходит менеджерам в DTF Telegram-бот.
- Встроить CTA на кастомный принт прямо в блок категорий на главной, а не отдельным баннером под ним.

## Визуальная идея
- Один цельный тёмный `studio`-макет с тёплым золотистым акцентом.
- Без англоязычных подписей, без набора равноправных лендинговых секций и без лишнего информационного шума.
- На главной CTA должна ощущаться частью того же семейства карточек категорий, но с более тёплым inset-блоком внутри.

## UX-структура страницы
1. Короткий вступительный блок:
   - что это за страница;
   - зачем нужна форма;
   - быстрый переход в Telegram для прямого диалога.
2. Пошаговая форма:
   - `Що вам потрібно?`
   - `На чому друкуємо?`
   - `Де буде принт?`
   - `Деталі макета`
   - `Параметри замовлення`
   - `Як з вами зв’язатися?`
3. Финальный блок:
   - ориентир по цене;
   - активные вторичные действия;
   - основной submit.

## Данные и backend
- Отдельная сущность лида: `CustomPrintLead`.
- Отдельная сущность файлов: `CustomPrintLeadAttachment`.
- Новый endpoint: `POST /custom-print/lead/`.
- Обработка через `forms.Form` с явной валидацией сценариев:
  - `ready` требует файл;
  - `adjust` требует файл и описание правок;
  - `design` требует brief, файл необязателен.
- После сохранения идёт уведомление в существующий DTF Telegram-бот.

## Главная страница
- Сетка категорий остаётся основной структурой.
- После карточек категорий добавляется широкая встроенная плитка:
  - кикер `Свій принт`;
  - заголовок `Замовити кастомний одяг`;
  - пояснение про футболки, худи и лонгсливы;
  - action `Перейти до заявки`.

## Ограничения итерации
- Конструктор пока остаётся недоступным и показывается как disabled-placeholder.
- Отправка заявки делается прямо с основной страницы через AJAX.
- Точный расчёт цены остаётся за менеджером; на странице даётся только ориентир.
