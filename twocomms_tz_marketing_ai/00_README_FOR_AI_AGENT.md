# TwoComms: общее ТЗ для AI-агента внедрения

## 1. Контекст задачи

Нужно улучшить сайт `twocomms.shop` не через полный редизайн, а через точное усиление позиционирования, воронки, текстов, SEO/GEO и доверия. На сайте уже есть сильная работа по дизайну, каталогу, кастомному конфигуратору и странице бренда. Нельзя ломать текущую структуру без причины. Главная задача — сделать так, чтобы пользователь быстрее понял:

1. что такое TwoComms;
2. чем бренд отличается от обычного магазина футболок;
3. почему одежда стоит внимания;
4. куда нажать дальше: каталог, кастом, история бренда или опрос;
5. почему бренду можно доверять.

## 2. Ключевое решение по позиционированию

Не привязывать бренд слишком сильно к Харькову на главной. Харьков важен как происхождение и доказательство характера, но не должен создавать ощущение локального магазина только для Харькова.

Правильный принцип:

> TwoComms — украинский streetwear / military-adjacent бренд с ветеранским основанием, собственным DTF-направлением и одеждой, в которой есть код, состояние и продолжение.

Харьков использовать как подтверждение:

> Народжений у Харкові. Зроблений для всієї України.

Не делать так:

> Харківський бренд одягу для харків’ян.

Почему: пользователь из Киева, Львова, Днепра, Одессы или другой страны должен чувствовать, что бренд для него тоже. Харьков должен усиливать историю, но не ограничивать рынок.

## 3. Главная стратегическая формула

TwoComms нельзя упаковывать просто как “ексклюзивні футболки, худі та лонгсліви”. Это слишком общее и легко теряется среди сотен магазинов.

Главная формула:

> Одяг з кодом продовження.

Расшифровка:

- не просто принт;
- не просто милитари;
- не агрессивный патриотический шум;
- не мерч ради мерча;
- одежда как состояние: собранность, внутренняя форма, движение дальше;
- бренд с реальной ветеранской историей и производственной базой.

## 4. Что нельзя делать

- Не превращать сайт в длинную “простыню” пафосного текста.
- Не писать искусственные фразы типа “інноваційний бренд нового покоління”.
- Не перегружать первый экран словами “ветеранский”, “Харьков”, “милитари”, “уникальный”, “премиальный” одновременно.
- Не ломать каталог и уже реализованные блоки.
- Не делать ощущение благотворительного проекта вместо бренда одежды.
- Не использовать Харьков как главный географический фильтр продаж.
- Не обещать то, что не подтверждено: количество стирок, часть прибыли на ВСУ, полный цикл производства в Харькове — всё это нужно проверять перед публикацией.

## 5. Что нужно сделать

1. Переписать первый экран главной.
2. Убрать противоречие в блоке опроса: проходить опрос можно не только зарегистрированным пользователям.
3. Добавить мягкую регистрацию после опроса для сохранения промокода и баллов.
4. Усилить воронку без полного редизайна.
5. Добавить блок доверия с внешними подтверждениями.
6. Усилить кастомный DTF как отдельное направление.
7. Убрать/заменить тестовый блог-пост `111`.
8. Создать SEO/GEO-контент-план.
9. Улучшить карточки товаров: смысл + доверие + уход + отзывы + фото.
10. Проверить техническое SEO, schema, индексируемость и аналитику.

## 6. Приоритеты внедрения

### Срочно, 1–3 дня

- Исправить блок опроса.
- Переписать hero на главной.
- Убрать или закрыть от индексации статью `111`.
- Добавить 3–4 trust-сигнала на главную.
- Проверить тексты, где Харьков звучит как ограничение, а не происхождение.

### Средний приоритет, 4–14 дней

- Усилить кастомную страницу.
- Пересобрать блоки главной в более логичную воронку.
- Добавить внутреннюю перелинковку.
- Создать 3 SEO-лендинга: ветеранский бренд, DTF-друк, мерч для команд/подразделений.

### Рост, 15–45 дней

- Запустить блог-кластер.
- Добавить schema-разметку.
- Добавить отзывы/UGC.
- Подключить события аналитики.
- Тестировать hero/CTA через A/B или поэтапные изменения.

## 7. Главное правило для AI-агента

В этих файлах даны рекомендуемые тексты. Их можно улучшать, но только если новый вариант:

1. звучит более живо;
2. не похож на AI-текст;
3. сильнее раскрывает бренд;
4. не теряет SEO-смысл;
5. не делает бренд узко харьковским;
6. не обещает неподтверждённые факты.

Если сомневаешься — используй предложенный текст без изменений.
