# Custom Print, DTF и B2B: ТЗ

## 1. Роль направления

Custom Print — одно из самых сильных направлений сайта. Его нужно продвигать как отдельный продукт, а не как дополнительную кнопку.

Позиция:

> Кастомний DTF-друк на одязі TwoComms: від однієї речі до дропу для команди, бренду або підрозділу.

## 2. Кому продаём

### Сегмент 1. Один человек

Потребность:

- хочу свою футболку/худи;
- хочу подарок;
- хочу принт на своей вещи;
- хочу не стандартный мерч.

Сообщение:

> Збери одну річ під себе: обери основу, колір, зону друку й завантаж макет.

### Сегмент 2. Команда/бренд

Потребность:

- небольшой дроп;
- мерч для команды;
- тест партии;
- быстро и без сложного производства.

Сообщение:

> Зробимо партію для бренду, команди або події: від макету до готових речей.

### Сегмент 3. Подразделение/волонтёры

Потребность:

- одежда с символом;
- подарок бойцам;
- вещи с логотипом/позывным;
- адекватная коммуникация.

Сообщение:

> Допоможемо зробити речі з вашим знаком: без зайвого шуму, з увагою до деталей і сенсу.

## 4. Что добавить на страницу

### 4.1. Блок “Як це працює”

1. Обираєш основу: футболка, худі, лонгслів або своя річ.
2. Вказуєш колір, розмір, кількість.
3. Обираєш зони друку.
4. Завантажуєш макет або описуєш ідею.
5. Отримуєш прорахунок і підтверджуєш запуск.

### 4.2. Блок “Для кого”

Карточки:

- Для себе
- Для подарунка
- Для бренду
- Для команди
- Для підрозділу
- Для волонтерського проєкту

### 4.3. Блок “Що можна друкувати”

- логотипи;
- ілюстрації;
- текстові принти;
- позивні;
- символи команди;
- дропи бренду;
- мінімалістичні знаки.

### 4.4. Блок “Макет”

Текст:

> Якщо макет уже готовий — завантаж його. Якщо є тільки ідея, напиши її в коментарі: підкажемо, як краще підготувати друк.

### 4.5. Блок FAQ

Вопросы:

- Чи можна замовити одну річ?
- Чи можна надрукувати на моєму одязі?
- Який файл потрібен для друку?
- Скільки часу займає замовлення?
- Чи робите партії для брендів?
- Чи можна зробити мерч для підрозділу?
- Як прати DTF-принт?

## 6. Микрокопирайтинг конфигуратора

### Для кого збираємо?

Вариант:

> Для себе, подарунка, команди або бренду — від цього залежить підбір основи, кількість і прорахунок.

### Завантажити макет

> PNG, PDF, SVG або інший файл. Якщо не впевнений у якості — завантаж, ми перевіримо.

### Комментарий

> Опиши ідею, розміри принта, бажаний ефект або дедлайн. Чим більше деталей — тим точніший прорахунок.

## 7. Прогноз

Страница может стать отдельным источником заявок и SEO-трафика. Особенно по запросам DTF-друк, друк на худі, кастомний одяг, мерч для команди.
