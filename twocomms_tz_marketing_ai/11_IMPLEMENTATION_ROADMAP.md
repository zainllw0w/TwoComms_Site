# Дорожная карта внедрения

## Этап 1. Срочные правки, 1–3 дня

### 1. Hero главной

- Заменить H1.
- Заменить subheadline.
- Добавить второй CTA на custom.
- Добавить microline/badges.

### 2. Опрос

- Убрать “лише для зареєстрованих користувачів”.
- Переписать блок.
- Добавить финальный экран промокода.
- Добавить мягкую регистрацию для сохранения.

### 3. Блог

- Удалить/закрыть от индексации `111`.
- Проверить sitemap после удаления.

### 4. Trust

- Добавить блок с внешними подтверждениями.
- Не перегружать главную.

## Этап 2. Усиление воронки, 4–10 дней

- Добавить блок быстрых маршрутов.
- Переписать заголовок и описание категорий.
- Усилить custom-блок на главной.
- Добавить внутренние ссылки между catalog/custom/about.
- Добавить события аналитики.

## Этап 3. Custom и товарные страницы, 10–20 дней

- Обновить custom hero.
- Добавить блок “як це працює”.
- Добавить FAQ на custom page.
- Добавить блок “для кого”.
- Обновить шаблон карточек товара.
- Добавить блок “хочеш інакше?” на товарах.

## Этап 4. SEO/GEO, 20–45 дней

- Создать первые 5 статей.
- Создать 3 лендинга:
  1. Ветеранський бренд одягу TwoComms
  2. Кастомний DTF-друк на одязі
  3. Мерч для команди, бренду або підрозділу
- Добавить schema.
- Проверить Merchant Center, если используется.
- Настроить Open Graph.

## Этап 5. Рост и тесты, 45–90 дней

- Добавить UGC.
- Запустить регулярный блог.
- Делать 1–2 внешних публикации в месяц.
- Тестировать hero/CTA.
- Анализировать конверсии.

## Что не делать в первом этапе

- Не переделывать полностью дизайн каталога.
- Не менять всю структуру сайта сразу.
- Не запускать 20 статей без качества.
- Не делать агрессивные попапы.
- Не перегружать главную длинной историей.

## Критерий успеха

После внедрения пользователь должен за первые 10 секунд понять:

1. TwoComms — украинский streetwear/military-adjacent бренд.
2. У бренда есть код и история.
3. Можно купить готовую вещь.
4. Можно создать свой принт.
5. Бренду можно доверять.
